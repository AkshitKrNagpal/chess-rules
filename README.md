> Chess Rules implementation as a standalone module.


## Install

```sh
$ npm install --save chess-rules
```


## Usage

```js
var chessRules = require('chess-rules');

chessRules('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
